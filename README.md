# 7日間スケジュール

## 概要

7日間スケジュールは、大学生活における予定管理・タスク管理・学習時間管理を一元化するために開発したWebアプリケーションです。

カレンダー機能とタスク管理機能を統合し、学習状況を可視化することで日々の自己管理を支援します。

## 開発背景

大学生活では、

- 授業予定
- 課題管理
- 学習時間の記録

を複数のアプリで管理していました。

そこで、自分に必要な機能を一つにまとめたオリジナルの管理ツールとして七色ダイアリーを開発しました。

## 主な機能

- 月表示カレンダー
- 週表示カレンダー
- 日表示カレンダー
- スケジュール登録
- タスク管理
- 学習時間記録
- 曜日ごとのテーマ変更

## 使用技術

- HTML
- CSS
- JavaScript
- LocalStorage
- Git

- GitHub Pages

## 工夫した点

### UIの整理

当初は1つのHTMLファイルに実装していましたが、保守性向上のため以下のように分割しました。

- index.html
- css/style.css
- js/main.js
- js/theme.js
- js/utils.js

### テーマ機能

曜日ごとに異なるテーマを設定し、利用時の楽しさを向上させました。

### カレンダー機能

ライブラリを使用せずJavaScriptのみで月・週・日表示を実装しました。
