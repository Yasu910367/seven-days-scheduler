# 7曜スケジュール

## 概要

7曜スケジュールは、大学生活における予定管理・タスク管理・学習時間管理を一元化するために開発したクラウド対応Webアプリケーションです。

Googleアカウントによるログイン機能とFirebase Firestoreによるクラウド保存機能を実装しており、PC・スマートフォン間でデータを同期できます。

予定管理、タスク管理、学習時間の記録を一つのアプリで完結させることで、日々の自己管理を支援します。

---

## 開発背景

大学生活では、

- 授業予定の管理
- 課題やタスクの管理
- 学習時間の記録

を複数のアプリで管理していました。

しかし、アプリを切り替えながら利用することに不便さを感じていたため、自分に必要な機能を一つにまとめた管理ツールとして「7曜スケジュール」を開発しました。

また、Webアプリ開発の学習を目的として、認証機能やクラウドデータベースを活用した実践的なアプリケーション開発にも挑戦しました。

## 工夫した点

### Firebaseによるクラウド同期

当初はLocalStorageを利用していましたが、複数端末間でデータを共有できないという課題がありました。

そこでFirebase AuthenticationとCloud Firestoreを導入し、ユーザーごとにデータを管理する仕組みを実装しました。

これにより、

- PCで登録した予定をスマホで確認
- スマホで記録した学習時間をPCで集計

といった利用が可能になりました。

### UIのモジュール化

保守性向上のため、機能ごとにJavaScriptファイルを分割しました。

```text
index.html
css/style.css
js/main.js
js/theme.js
js/utils.js
js/firebase.js
```

### カレンダー機能

外部ライブラリを使用せず、JavaScriptのみで

- 月表示
- 週表示
- 日表示

を実装しました。

### Gitブランチ運用

Firestore移行作業では、

```text

main
↓
feature/schedule-firestore
↓
Pull Request
↓
Merge
```

という開発フローを採用し、安全に機能追加を行いました。
